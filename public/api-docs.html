API Documentation

Integrate your Minecraft server with Ender List in minutes. Every server gets a unique API token when listed.

Getting Your API Token

When you add your server, you'll receive a token starting with el_. Pass it via the X-API-Token header in all requests.

Base URL

https://enderlist.com/api/v1

Authentication

X-API-Token: el_your_token_here

Endpoints

GET /api/v1/server

Get your server's current info.

curl -H "X-API-Token: el_xxx" https://enderlist.com/api/v1/server

POST /api/v1/server/heartbeat

Update player count and online status. Send this every 1-5 minutes from your plugin.

curl -X POST https://enderlist.com/api/v1/server/heartbeat \
  -H "X-API-Token: el_xxx" \
  -H "Content-Type: application/json" \
  -d '{"player_count": 42, "max_players": 100, "is_online": true}'

GET /api/v1/server/votes

Get recent votes. Optional ?since=ISO_DATE param (defaults to last 24h).

curl -H "X-API-Token: el_xxx" https://enderlist.com/api/v1/server/votes

POST /api/v1/server/vote/check

Check if a specific player has voted recently. Great for in-game vote rewards!

curl -X POST https://enderlist.com/api/v1/server/vote/check \
  -H "X-API-Token: el_xxx" \
  -H "Content-Type: application/json" \
  -d '{"username": "Notch"}'

Response:

{"has_voted": true, "username": "Notch"}

GET /api/v1/server/top-voters

Get the top 10 voters for your server this month.

curl -H "X-API-Token: el_xxx" https://enderlist.com/api/v1/server/top-voters

GET /api/v1/server/metrics

Get player count history. Optional ?hours=24 (max 168 = 7 days).

curl -H "X-API-Token: el_xxx" "https://enderlist.com/api/v1/server/metrics?hours=48"

Rate Limits

API requests are limited to 120 requests per minute. Heartbeats should be sent every 1-5 minutes (not more frequently).

Spigot / Paper Plugin Example

Minimal Java snippet for sending heartbeats:

// In your plugin's onEnable() — schedule a repeating task
Bukkit.getScheduler().runTaskTimerAsynchronously(this, () -> {
    try {
        URL url = new URL("https://enderlist.com/api/v1/server/heartbeat");
        HttpURLConnection con = (HttpURLConnection) url.openConnection();
        con.setRequestMethod("POST");
        con.setRequestProperty("X-API-Token", "el_YOUR_TOKEN");
        con.setRequestProperty("Content-Type", "application/json");
        con.setDoOutput(true);

        String json = String.format(
            "{"player_count":%d,"max_players":%d,"is_online":true}",
            Bukkit.getOnlinePlayers().size(),
            Bukkit.getMaxPlayers()
        );

        con.getOutputStream().write(json.getBytes());
        con.getInputStream().close();
    } catch (Exception e) { e.printStackTrace(); }
}, 0L, 6000L); // Every 5 minutes (6000 ticks)

Error Codes

  • 401 — Missing or invalid API token
  • 403 — Server is banned
  • 429 — Rate limit exceeded
  • 500 — Internal server error
