Frequently Asked Questions

How does voting work?

Voting is how the Ender List community helps surface the best servers. When you vote for a server, it boosts that server's ranking in the list. Here's how it works:

  • Sign in using Google or Discord
  • Find a server you enjoy playing on
  • Click "Vote for this Server" on the server's detail page
  • You can vote once every 2 hours per server
  • Votes accumulate over time — servers with the most votes rank higher

Server owners can also reward voters by using our API to detect when a player votes and grant in-game rewards.

How do I add my server?

Sign in, click "+ Add Server", and fill out the form with your server details. You'll receive an API token immediately that you can use to integrate with our tracking system.

What is the API token?

When you add a server, you're given a unique API token (starts with el_). Use this token to send heartbeat data (player counts, online status) and check vote status from your Minecraft plugin. See our API documentation for full details.

What does "Featured" / "VIP" mean?

Featured servers appear at the top of the list with a purple VIP badge. This is managed by the Ender List staff team and lasts for 30 days. Interested? Reach out via our advertiser page.

How is ranking determined?

By default, servers are ranked by total votes. You can also sort by most players online or newest. Featured servers always appear at the top regardless of sort order.

Can I edit my server after adding it?

Yes. Sign in with the same account you used to add the server. You'll be able to update the name, description, tags, version, and links.

Why was my server removed or banned?

Servers may be removed for violating our Terms of Service, including but not limited to: vote manipulation, misleading server info, illegal content, or abusive behavior. If you believe this was in error, contact us.

Is Ender List affiliated with Mojang?

No. Ender List is an independent, community-run Minecraft server listing site. We are not affiliated with, endorsed by, or connected to Mojang Studios or Microsoft.
